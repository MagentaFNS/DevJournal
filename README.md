## About the project

Дневник разработчика с аналитикой - ты и твоя команда каждый день пишете короткий отчёт: «Что изучил, что сломал, что починил». Бэкенд анализирует ваши записи и показывает прогресс: график коммитов, частоту ошибок, количество нового кода.

## Установка DevJournal

**macOS / Linux**

```bash
git clone https://github.com/MagentaFNS/devjournal.git
cd devjournal
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

**Windows (PowerShell)**

```powershell
git clone https://github.com/MagentaFNS/devjournal.git
cd devjournal
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Overview

- **Простая настройка, масштабируемость по требованию** - Начните работу в локальном режиме за считанные минуты. Затем расширяйте масштаб настолько, насколько вам нужно, и так быстро, как вам нужно.
